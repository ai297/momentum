Инструкция по проверке

  • Базовая функциональность
    • (выполнено)время выводится в 24-часовом формате, обновляется каждую секунду
    • (выполнено)выводится день недели, дата, месяц, например: "Пятница, 25 ноября"
    • (выполнено)можно ввести имя пользователя и его цель. Для ввода данных используется клавиша Enter
    • (выполнено)имя пользователя и его цель сохраняются в local storage и отображаются после обновления страницы
    • (выполнено)фоновое изображение и приветствие изменяются в зависимости от времени суток (утро, день, вечер, ночь)
    • (выполнено)при клике в поле ввода текст, который там был, исчезает, если пользователь ничего не ввёл или ввёл пустую строку, текст восстанавливается
  • Смена фонового изображения
    • (выполнено)фоновые изображения меняются каждый час и выбираются в зависимости от времени суток (утро, день, вечер, ночь)
    • (выполнено)есть кнопка, при клике по которой можно пролистать все фоновые изображения за сутки
    • (выполнено)изображения пролистываются в том же порядке, в котором они менялись бы в реальном времени (утро, день, вечер, ночь), начиная с того, которое соотвествует текущему времени суток
    • (выполнено)при обновлении страницы формируется новый список фоновых изображений на текущие сутки
    • (выполнено)при клике по кнопке для обновления фонового изображения происходит плавная смена фоновых изображений, нет ситуации, когда пользователь видит частично загрузившееся изображение
  • Цитата дня
    • (выполнено)при загрузке приложения выводится цитата или другой короткий текст (высказывание, шутка, анекдот и т.д.)
    • (выполнено)при перезагрузке страницы цитата заменяется на другую
    • (выполнено)есть кнопка, при клике по которой меняется цитата
  • Прогноз погоды
    • (выполнено)в приложении есть возможность указать город
    • (выполнено)для указанного пользователем города выводится прогноз погоды
    • (выполнено)указанный пользователем город сохраняется в local storage и отображается при обновлении страницы. Также отображается прогноз погоды для него
    • (выполнено)прогноз погоды включает в себя данные о температуре, относительной влажности воздуха, скорости ветра
    • (выполнено)прогноз погоды включает в себя иконку погоды
    • (выполнено)если пользователь вводит пустую строку, данные не меняются, отображается прежний прогноз погоды. Если пользователь вводит данные, для которых API погоды не возвращает результат, выводится уведомление об ошибке в человекочитаемом формате
  • Адаптивный дизайн
    • (выполнено)элемены приложения оптимально занимают площадь страницы
    • (выполнено)приложение корректно отображается как на компьютере, так и на мобильных устройствах. Отсутствует горизонтальная полоса прокрутки. Минимальное разрешение экрана, при котором проверяем корректность отображения приложения - 320px
    • (выполнено)при смене даты и времени, смене цитаты, смене фонового изображения, содержание страницы не дёргается, размеры блоков остаются прежними, кнопки для смены цитаты и смены фонового изображения остаются на своих местах
    • (выполнено)шрифт достаточного размера (не меньше 14рх) и контрастности Colour contrast checking tool, хорошо читается на любом фоне. Можно использовать полупрозрачный overlay (есть в коллекции изображений) или другой способ, позволяющий обеспечить читаемость текста.
    • (выполнено)интерактивность кликабельных элементов не сводится к свойству cursor:pointer. Стили при наведении, фокусе, клике меняются плавно (используйте свойство transition или @keyframes)
  • каждый выполненный пункт - +2 балла
  • Обратите внимание. Если прогноз погоды или смена цитаты не работают, посмотрите в консоль браузера. Код ошибки 403 или 501 говорит о проблеме с доступом к API. В таком случае приложение следует проверить позже - через несколько часов или на следующий день. Баллы при этом снижать не нужно, это проблема сервиса, а не приложения. Чтобы избежать подобных проблем, авторам работ не рекомендуется выкладывать их в общий доступ до окончания кросс-чека.
